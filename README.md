# BOOK COMMUNITY - WEBSITE

## REQUIREMENT
- Docker

## USAGE

### FOR WINDOWS
1. Run command in folder
``` 
copy .env.example .env
copy app/.env.example app/.env
docker-compose up
```

### FOR MACOS/LINUX
1. Run command in folder
``` 
cp .env.example .env
cp app/.env.example app/.env
docker-compose up
```
